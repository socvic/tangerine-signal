# Design Decisions

## Why Stacks?

Stacks provides unique advantages:
- Bitcoin security through Proof of Transfer
- Clarity language for predictable smart contracts
- Low transaction costs compared to alternatives

## Why Clarity?

Clarity was chosen for smart contracts because:
- Decidable - you can know what a program will do before executing it
- No Turing-completeness means fewer security risks
- Built-in support for Bitcoin state reads

## Why React + Vite?

- Fast development experience with HMR
- Optimized production builds
- Strong TypeScript support
