# Frequently Asked Questions

## General

**What is Tangerine Signal?**
Tangerine Signal is a decentralized micro-polling application built on the Stacks blockchain.

**How does it work?**
Users can create polls and vote on them through Clarity smart contracts, with all data stored on-chain.

## Technical

**What blockchain does it use?**
Stacks blockchain, which provides smart contract functionality with Bitcoin security.

**What is Clarity?**
Clarity is a decidable smart contract language designed for the Stacks blockchain.

**How are polls stored?**
Polls are stored in on-chain maps within the micro-polls Clarity contract.
