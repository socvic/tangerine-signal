# Error Codes

## Contract Errors

| Code | Constant | Description |
|------|----------|-------------|
| u100 | ERR-POLL-NOT-FOUND | Poll does not exist |
| u101 | ERR-POLL-INACTIVE | Poll is no longer active |
| u102 | ERR-ALREADY-VOTED | User has already voted on this poll |
| u103 | ERR-INVALID-OPTION | Option index is out of range |
| u104 | ERR-POLL-FULL | Maximum number of options reached |

## Frontend Errors

- `WALLET_CONNECTION_FAILED`: Could not connect to wallet
- `CONTRACT_CALL_FAILED`: Transaction submission failed
- `NETWORK_ERROR`: Network connectivity issue
