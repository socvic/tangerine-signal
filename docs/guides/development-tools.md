# Development Tools

## Required

- **Clarinet**: Smart contract development and testing
- **Node.js 18+**: JavaScript runtime
- **npm**: Package manager
- **Git**: Version control

## Recommended

- **VS Code**: IDE with Clarity extension
- **Hiro Wallet**: Browser wallet for testing
- **Stacks Explorer**: View on-chain data

## Installation

```bash
# Install Clarinet
curl -sL1 https://hirosystems.z19.web.core.windows.net/clarinet/install.sh | sh

# Install Node.js (using nvm)
nvm install 18

# Install dependencies
npm install
```
