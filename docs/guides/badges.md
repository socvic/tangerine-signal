# Project Badges

## Recommended Badges

- Build status
- Test coverage
- License
- Version
- Stack

## Adding Badges

Badges can be added to the main README.md using shields.io:

```markdown
![Build Status](https://img.shields.io/github/workflow/status/user/repo/build)
![License](https://img.shields.io/github/license/user/repo)
```
