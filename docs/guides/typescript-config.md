# TypeScript Configuration

## Root Configuration

The root `tsconfig.json` provides base settings for the project.

## Frontend Configuration

The frontend has three TypeScript configs:
- `tsconfig.json` - Base references
- `tsconfig.app.json` - Application code settings
- `tsconfig.node.json` - Node/Vite tooling settings

## Key Settings

- Strict mode enabled
- ES module interop
- Path aliases configured
