# Network Configuration

## Devnet

Local development network configured in `settings/Devnet.toml`.

## Testnet

Stacks testnet for staging deployments. Configured in `settings/Testnet.toml`.

## Mainnet

Production network. Requires careful review before deployment. Configured in `settings/Mainnet.toml`.

## Switching Networks

Update the `STACKS_NETWORK` environment variable to switch between networks.
