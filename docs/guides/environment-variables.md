# Environment Variables

## Backend

| Variable | Description | Default |
|----------|-------------|---------|
| `STACKS_NETWORK` | Target network | `devnet` |

## Frontend

| Variable | Description | Default |
|----------|-------------|---------|
| `VITE_API_URL` | API endpoint URL | `http://localhost:3000` |
| `VITE_CONTRACT_ADDRESS` | Contract address | - |
