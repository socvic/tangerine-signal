# Build Configuration

## Vite Configuration

The project uses Vite with the following key settings:

- React plugin for JSX transformation
- Path aliases for clean imports
- Environment variable handling with VITE_ prefix

## Build Output

Build output is placed in the `dist/` directory.

## Environment Variables

- `VITE_CONTRACT_ADDRESS`: The deployed contract address
- `VITE_API_URL`: Backend API endpoint
